== Description ==

This is just a funny little plugin that displays lyrics from Eminems 'My name is'. Just like the popular Plugin 'Hello Dolly' but with Slim Shady flow.
But be careful: swear words are used!

== Installation ==

Upload the Lyrics by Eminem plugin to your blog, activate it and that's it. 
Enjoy the best of Eminem while working.
